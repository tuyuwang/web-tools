# 工具网站 (Tool Website)

一个功能丰富的在线工具网站，提供各种实用的工具，完全前端实现，保护用户隐私。

## 🚀 新增功能

### PDF工具套件
- **PDF合并** - 将多个PDF文件合并为一个，支持拖拽排序
- **PDF分割** - 将PDF文件分割成多个文件，支持按页数分割
- **图片转PDF** - 将多张图片合并成PDF文件，支持页面大小设置
- **PDF转图片** - 将PDF页面转换为PNG/JPG图片
- **PDF旋转** - 旋转PDF页面，支持90度、180度旋转
- **PDF加密/解密** - 为PDF添加密码保护或移除密码

### 高级图像处理
- **背景移除** - 智能移除图片背景，适用于纯色背景
- **在线图片编辑器** - 功能强大的图片编辑工具，支持滤镜、旋转、翻转
- **图片压缩** - 优化图片大小，支持质量调节
- **图片格式转换** - 支持多种图片格式间的高质量转换

### 文本处理工具
- **OCR文字识别** - 从图片中识别和提取文字，支持多种语言
- **文字转语音** - 将文字转换为语音，支持多种语言和声音
- **文本分析** - 分析文本的字符数、词数、行数等统计信息
- **文本比较** - 比较两段文本的差异

### 生产力工具
- **URL短链生成器** - 生成短链接，支持自定义短码和二维码
- **在线白板** - 协作白板工具，支持绘图、便签、图形
- **发票生成器** - 生成专业发票，支持自定义模板
- **屏幕录制** - 录制屏幕内容，支持录制窗口或整个屏幕

## 🛠️ 技术栈

- **前端框架**: Next.js 14
- **UI库**: Tailwind CSS
- **PDF处理**: pdf-lib
- **图像处理**: Canvas API
- **OCR识别**: tesseract.js
- **文件处理**: react-dropzone
- **二维码生成**: qrcode
- **语音合成**: Web Speech API

## ✨ 特性

- **完全前端处理** - 所有工具都在浏览器中运行，文件不上传到服务器
- **隐私保护** - 用户数据完全本地处理，确保隐私安全
- **响应式设计** - 支持桌面端和移动端，自适应各种屏幕尺寸
- **多语言支持** - 支持中文和英文界面
- **PWA支持** - 支持安装为桌面应用
- **暗色主题** - 支持明亮和暗色两种主题模式

## 🚀 快速开始

### 安装依赖
```bash
cd Tool
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 静态导出
```bash
npm run build:static
```

## 🌟 亮点功能

### 完全客户端处理
所有工具都在用户浏览器中运行，不需要将文件上传到服务器，确保：
- **隐私安全** - 文件不离开用户设备
- **快速处理** - 无需网络传输，处理速度更快
- **离线可用** - 部分功能支持离线使用

### 专业级PDF处理
使用pdf-lib库实现专业级PDF处理功能：
- 支持合并多个PDF文件
- 支持按页数分割PDF
- 支持图片转PDF，可设置页面大小
- 支持PDF加密和解密

### 先进的图像处理
结合Canvas API实现高质量图像处理：
- 智能背景移除算法
- 实时滤镜预览
- 无损图片格式转换
- 专业级图片编辑功能

### 多语言OCR识别
集成tesseract.js实现强大的OCR功能：
- 支持中文、英文、日文等多种语言
- 实时进度显示
- 可编辑识别结果
- 支持下载文本文件

## 📱 PWA支持

本网站支持Progressive Web App (PWA)功能：
- 可安装到桌面或主屏幕
- 支持离线缓存
- 类似原生应用的体验

## 🔧 配置说明

### 环境变量
复制 `env.local.template` 为 `.env.local` 并配置相关参数：
```bash
cp env.local.template .env.local
```

### Supabase配置（可选）
如果需要反馈功能，请配置Supabase：
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
