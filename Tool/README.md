# Tool 工具集合

高可用、高访问的静态工具网站，提供各种实用的在线工具。

## 🚀 最新更新

### 媒体工具重大简化 (最新)

我们对媒体工具进行了**全面简化**，移除了虚假和误导性的功能，专注于提供**真实可用**的前端功能：

#### ✅ 新增的真实功能
- **音频播放器**：完整的音频播放控制、音频可视化、文件管理
- **视频播放器**：原生视频播放、全屏支持、基本信息显示
- **媒体信息查看器**：真实的元数据提取和显示

#### ❌ 移除的虚假功能
- 假的音视频格式转换（只改文件扩展名）
- 误导性的转换进度条
- 无效的音频质量设置
- 复杂的多音轨处理（前端API限制）

#### 🎯 设计原则
- **诚实**：只提供前端真正能实现的功能
- **实用**：专注于用户真正需要的播放和查看功能
- **简洁**：移除复杂无用的假功能

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装步骤

```bash
# 克隆项目
git clone <repository-url>
cd Tool

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 可用脚本

```bash
# 开发相关
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
npm run start        # 启动生产服务器

# 代码质量
npm run lint         # ESLint检查
npm run format       # Prettier格式化
npm run type-check   # TypeScript类型检查

# 测试相关
npm run test         # 运行测试
npm run test:watch   # 监视模式测试
npm run test:coverage # 测试覆盖率

# 构建优化
npm run build:static    # 静态化构建
npm run build:optimized # 优化构建
npm run analyze         # 构建分析
```
